RectangleCommand Methods

Methods

CheckParameters Validates that the Rectangle command receives exactly two parameters (width and height). The dimensions can be literal numbers or variable names.
Compile Compiles the Rectangle command. No compilation is needed for simple drawing commands.
Execute Executes the Rectangle command by drawing a rectangle outline at the current pen position. Re-evaluates dimensions at execution time to support variable changes.
Set Sets up the Rectangle command with parameters and performs validation.
ValidateDimensions Validates and evaluates the dimension parameters. Handles both literal numbers and variable references.
