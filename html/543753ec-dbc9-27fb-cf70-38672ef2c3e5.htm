MyCommandFactory Methods

Methods

Instance Gets the singleton instance of MyCommandFactory. Creates a new instance on the first call; subsequent calls return the existing instance.
IsLikelyVariableName Determines if a command name looks like it could be a variable name rather than a BOOSE command. This helps avoid unnecessary calls to the base factory for likely variable references.
MakeCommand Creates and returns a command instance of the specified type. Attempts to create custom commands (Clear, Reset, Circle, MoveTo, Rectangle, Int, Real, Array, Write, Poke, Peek, If, Else, Endif, While, Endwhile) first, then delegates to the base factory for built-in BOOSE commands. Also handles variable assignments by treating them as Int commands.
