MyCommandFactoryMakeCommand Method

Creates and returns a command instance of the specified type. Attempts to create custom commands (Clear, Reset, Circle, MoveTo, Rectangle, Int, Real, Array, Write, Poke, Peek, If, Else, Endif, While, Endwhile) first, then delegates to the base factory for built-in BOOSE commands. Also handles variable assignments by treating them as Int commands.

Parameters

commandType  String
The type of command to create. Case-insensitive. Recognized custom types: "clear", "reset", "circle", "moveto", "rect", "int", "real", "array", "write", "poke", "peek", "if", "else", "endif", "while", "endwhile", "end". If commandType is not a recognized command, it may be a variable name for assignment (e.g., width = 100). Other types are delegated to the base CommandFactory.

Return Value

ICommand
An ICommand instance corresponding to the specified command type.

Remarks

Command resolution order: 1. Check for custom "clear" command 2. Check for custom "reset" command 3. Check for custom "circle" command 4. Check for custom "moveto" command 5. Check for custom "rect" command 6. Check for custom "int" command 7. Check for custom "real" command 8. Check for custom "array" command 9. Check for custom "write" command 10. Check for custom "poke" command 11. Check for custom "peek" command 12. Check for custom "if" command 13. Check for custom "else" command 14. Check for custom "endif" or "end if" command 15. Check for custom "while" command 16. Check for custom "endwhile" or "end while" command 17. Delegate to base factory for built-in commands 18. If no match, return an AppInt instance (handles both declarations and assignments)

Exceptions

CommandException Thrown when: - The canvas is null for custom commands (Clear, Reset, Circle, MoveTo, Rectangle, Write). - An unknown command type is requested and the base factory returns null.
