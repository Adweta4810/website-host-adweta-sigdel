WriteCommand Class

Represents the write command in the BOOSE language. This command outputs text to the canvas or application output area, allowing messages, variable values, and program feedback to be displayed.

Methods

CheckParameters Validates that the Write command receives at least one parameter (the expression). The expression can be a literal number, a variable name, or an arithmetic expression.
Compile Compiles the Write command. No compilation is needed for write commands.
EvaluateExpression Evaluates an expression that may contain variables and arithmetic operators. Also supports string concatenation with the + operator (e.g., "prefix" + variable). Supports operators: +, -, *, /, % with standard precedence.
EvaluateStringExpression Evaluates string concatenation expressions with the + operator. Supports mixing string literals (in quotes) with variable references. Example: "prefix" + variable + "suffix"
Execute Executes the Write command by evaluating the expression and writing the result to the canvas. Evaluates expressions at execution time to support variable changes.
Set Sets up the Write command with the program context and parameters. Handles both literal values and expressions with variables.

Fields

_canvas The canvas instance on which the text will be written.
_expression The expression string to evaluate and write (may contain variables and arithmetic operations).
_program The stored program context used to evaluate variable expressions.
