AppCanvas Methods

Methods

Circle Draws or fills a circle at the current pen position with the specified radius.
Clear Clears the canvas by filling it with white color.
DrawTo Draws a line from the current pen position to the specified coordinates and updates the pen position.
getBitmap Gets the canvas bitmap for display or export.
MoveTo Moves the pen to the specified coordinates without drawing a line.
Rect Draws or fills a rectangle at the current pen position with the specified dimensions.
Reset Resets the canvas to its initial state. This moves the pen to the starting position (100, 100) and clears the drawing surface. BOOSE specification: reset clears the canvas and repositions the pen.
Set Resizes the canvas to the specified dimensions.
SetBgColour Sets the background color using RGB values. BOOSE supports the 'bgcolor r g b' command for this operation.
SetColour Sets the pen color using RGB values. BOOSE supports the 'colour r g b' command for this operation.
SetDrawingColor Sets the drawing color for both pen and brush.
Tri Draws or fills a triangle at the current pen position with the specified dimensions. The triangle's base is horizontal with the peak pointing upward.
WriteText Writes text on the canvas at the current pen position.
