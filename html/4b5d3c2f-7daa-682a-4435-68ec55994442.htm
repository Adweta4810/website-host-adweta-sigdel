EndWhileCommand Methods

Methods

CheckParameters Validates the Endwhile command parameters. The Endwhile command typically takes no parameters, but may receive "while" when the syntax is "end while".
Compile Compiles the Endwhile command to find matching while.
Execute Executes the Endwhile command by jumping back to the while statement to re-evaluate the condition, or continuing past if the condition is false.
GetEndWhileIndex Gets the endwhile index.
GetWhileIndex Gets the while index.
Set Sets up the Endwhile command with the program context and parameters.
SetEndWhileIndex Sets the endwhile index.
SetWhileIndex Sets the while index.
