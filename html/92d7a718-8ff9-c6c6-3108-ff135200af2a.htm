ArrayCommand Fields

Fields

_arrayName The array variable name.
_arrayTypes A static dictionary to track array types (int or real) for poke/peek commands.
_columns The second dimension (columns) of the array. Default is 1 for 1D arrays.
_columnsParam Column size parameter string (may contain expressions).
_intArrays A static dictionary to store integer array values. Stores 2D arrays as dictionary with key "varName"
_program The stored program context where variables will be registered.
_realArrays A static dictionary to store real array values with full precision. Stores 2D arrays as dictionary with key "varName[row,col]"
_rows The first dimension (rows) of the array.
_rowsParam Row size parameter string (may contain expressions).
_type The array type: "int" or "real"
arrayCount A static variable to track the number of declared array variables.
