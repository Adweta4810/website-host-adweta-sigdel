RectangleCommand Class

Represents the rectangle drawing command in the BOOSE language. This command draws a rectangle on the canvas using the current pen position, width, height, and pen colour.

RectangleCommand Initializes a new instance of the AppRectangle class with a canvas reference.

Methods

CheckParameters Validates that the Rectangle command receives exactly two parameters (width and height). The dimensions can be literal numbers or variable names.
Compile Compiles the Rectangle command. No compilation is needed for simple drawing commands.
Execute Executes the Rectangle command by drawing a rectangle outline at the current pen position. Re-evaluates dimensions at execution time to support variable changes.
Set Sets up the Rectangle command with parameters and performs validation.
ValidateDimensions Validates and evaluates the dimension parameters. Handles both literal numbers and variable references.

Fields

_canvas The canvas instance on which the rectangle will be drawn.
_height The height of the rectangle, validated and stored during parameter checking.
_heightParam The height parameter string (may contain variable names).
_program The stored program context used to evaluate variable expressions.
_width The width of the rectangle, validated and stored during parameter checking.
_widthParam The width parameter string (may contain variable names).
