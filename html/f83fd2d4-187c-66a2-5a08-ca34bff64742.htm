CircleCommand Class

Represents the circle drawing command in the BOOSE language. This command is responsible for drawing a circle on the canvas using the current pen colour and position.

Constructors

CircleCommand Initializes a new instance of the AppCircle class with a canvas reference.

Methods

CheckParameters Validates that the Circle command receives exactly one parameter (the radius). The radius can be a literal number or a variable name or an expression.
Compile Compiles the Circle command. No compilation is needed for simple drawing commands.
Execute Executes the Circle command by drawing a circle outline at the current pen position. Re-evaluates the radius at execution time to support variable changes.
Set Sets up the Circle command with parameters and performs validation.
ValidateRadius Validates and evaluates the radius parameter. Handles both literal numbers, variable references, and arithmetic expressions.

Fields

_canvas The canvas instance on which the circle will be drawn.
_program The stored program context used to evaluate variable expressions.
_radius The radius of the circle, validated and stored during parameter checking.
_radiusParam The radius parameter string (may contain variable names).
