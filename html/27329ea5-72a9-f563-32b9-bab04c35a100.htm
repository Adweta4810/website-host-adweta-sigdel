EndIfCommand Class

Represents the endif control command in the BOOSE language. This command marks the end of an if conditional block and signals the program to resume normal sequential execution.

Methods

CheckParameters Validates the Endif command parameters. The Endif command typically takes no parameters, but may receive "if", "while", or "for" when the syntax is "end if", "end while", or "end for".
Compile Compiles the Endif command. No compilation needed for endif statements.
Execute Executes the Endif command by popping the conditional context. If this was actually an "end while" or "end for" command, delegates to AppEndWhile/AppEndFor.
Set Sets up the Endif command with the program context and parameters. If the parameter is "while" (indicating "end while" syntax), delegates to AppEndWhile. If the parameter is "for" (indicating "end for" syntax), delegates to AppEndFor.

Fields

_delegatedCommand The actual command to execute (either this or a delegated AppEndWhile/AppEndFor).
_program The stored program context.
