ExecutionStateManager Methods

Methods

GetDepth Gets the current nesting depth of conditional blocks.

PopContext Pops the conditional context when endif is encountered.
PushContext Pushes a new conditional context when an if statement is encountered.
Reset Resets the execution state (useful for testing or restarting programs).
ShouldSkipExecution Checks whether commands should be skipped based on current conditional state.
