MyCommandFactory Class

Factory class for creating BOOSE command instances. Implements the Factory design pattern by extending CommandFactory to provide creation of custom commands (Clear, Reset, Circle, MoveTo, Rectangle) while delegating other commands to the base factory. Uses the singleton pattern to ensure a single factory instance throughout the application.

Remarks

This factory maintains a reference to an ICanvas instance that is used to initialize all created commands. Custom commands are checked first before delegating to the base factory for built-in BOOSE commands.

Attempts to create custom commands (Clear, Reset, Circle, MoveTo, Rectangle, Int, Real, Array, Write, Poke, Peek, If, Else, Endif, While, Endwhile) first, then delegates to the base factory for built-in BOOSE commands.
