WhileCommand Methods

Methods

CheckParameters Validates that the While command receives a condition parameter.
Compile Compiles the While command to find matching endwhile.
EvaluateCondition  
EvaluateExpression Evaluates an expression that may contain variables and arithmetic operators. Supports operators: +, -, *, /, % with standard precedence.
Execute Executes the While command by evaluating the condition. If true, continues to the next command (loop body). If false, jumps to the command after endwhile. Note: The actual loop control (jumping back/forward) is handled by Form1.ExecuteProgramWithLoops()
GetCondition Gets the condition for this while loop.
GetEndWhileIndex Gets the endwhile index for this while loop.
Set Sets up the While command with the program context and parameters.
SetEndWhileIndex Sets the endwhile index.
SetWhileIndex Sets the while index in the program.
