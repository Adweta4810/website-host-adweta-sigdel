RealCommand Class

Represents the real variable declaration command in the BOOSE language. This command is used to declare and initialise floating-point (real) variables that can be used in expressions and calculations during program execution.

Methods

CheckParameters Validates that a real variable name is provided and checks restrictions. Also handles optional initialization values (e.g., "radius" or "width = 2.5*radius"). For assignments to existing variables, validation happens in Execute() method.
Compile Compiles the Real command. No compilation is needed for variable declaration.
EvaluateArithmeticExpression Evaluates an expression that may contain variables and arithmetic operators. Supports operators: +, -, *, /, % with standard precedence.
Execute Executes the Real command by registering the variable in the stored program or updating it if it already exists. If an initial value was provided, it will be evaluated and assigned. This supports both declarations (real radius) and assignments (radius = 50.5).
ResetVariableCount Resets the variable counter. Useful for testing or restarting the program.
Restrictions Restricts the number of real variables to be no more than 100. Throws a RestrictionException if the count would exceed 100.
Set Sets up the Real command with the program context and parameters. Handles both simple declarations (real radius) and initializations (real radius = 50.5). Also handles assignments (width = 2.5*radius where width is the command name).
SetRealVariable Sets the value of a real variable in the dictionary. This allows other commands like AppPeek to update real variable values with full precision.
SetVariableName Sets the variable name explicitly. Used for assignments where the variable name is the command type and not included in the parameters.
TryGetRealVariable Gets the value of a real variable if it exists in the dictionary. This allows other classes like AppWrite to access real variable values with full precision.

Fields

_initialValue The initial value expression for the variable (can be a literal number or an expression).
_isAssignment Indicates whether this is an assignment operation (has = operator in params).
_program The stored program context where variables will be registered.
_realVariables A static dictionary to store real variable values with full precision. This works around the limitation of Evaluation.Value being an int.
_variableName The variable name to be declared or assigned.
variablenum A static variable to track the number of declared real variables.
