IntCommand Methods

Methods

CheckParameters Validates that an integer variable name is provided and checks restrictions. Also handles optional initialization values (e.g., "radius" or "width = 2*radius"). For assignments to existing variables, validation happens in Execute() method.
Compile Compiles the Int command. No compilation is needed for variable declaration.
EvaluateArithmeticExpression Evaluates an expression that may contain variables and arithmetic operators. Supports operators: +, -, *, /, % with standard precedence. Checks local scope first, then global scope.
Execute Executes the Int command by registering the variable in the stored program or updating it if it already exists. If an initial value was provided, it will be evaluated and assigned. This supports both declarations (int radius) and assignments (radius = 50).
ResetVariableCount Resets the variable counter. Useful for testing or restarting the program.
Restrictions Restricts the number of integer variables to be no more than 100. Throws a RestrictionException if the count would exceed 100.
Set Sets up the Int command with the program context and parameters. Handles both simple declarations (int radius) and initializations (int radius = 50). Also handles assignments (width = 2*radius where width is the command name).
SetLocalScope Sets the local scope for variable lookup during method execution. When set, variables are looked up in this scope first before global scope.
SetVariableName Sets the variable name explicitly. Used for assignments where the variable name is the command type and not included in the parameters.
