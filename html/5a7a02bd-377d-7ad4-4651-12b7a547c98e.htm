ArrayCommand Class

Implements the BOOSE Array command to declare arrays of integers or reals. Extends BOOSE.Array to provide compatibility with BOOSE's Peek and Poke commands. Syntax: array [type] [name] [size] [optional dimensions] Example: array int myArray 10,2 (creates 10x2 array) array real prices 10 (creates 1D array of size 10)
